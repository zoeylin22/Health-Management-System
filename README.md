# 健康管理系統（Health Management System）

## 專案簡介

本專案為「系統分析與設計」課程之系統分析專案，建立一套健康管理系統，協助使用者記錄個人健康資訊、運動紀錄與健康目標，並透過系統化方式進行健康管理與分析。

---

## 專案目標

* 提供會員註冊與登入功能
* 提供健康資料紀錄功能
* 提供運動紀錄管理功能
* 提供 BMI 自動計算功能
* 提供健康分析功能
* 提供健康目標設定功能

---

## 系統功能模組

### 會員管理

* 會員註冊
* 會員登入
* 個人資料管理
* 密碼修改

### 健康資料管理

* 新增健康紀錄
* 修改健康紀錄
* 查詢健康紀錄
* BMI 計算

### 運動管理

* 新增運動紀錄
* 修改運動紀錄
* 查詢運動紀錄
* 熱量消耗紀錄

### 健康分析

* BMI 趨勢分析
* 體重變化分析
* 運動統計分析
