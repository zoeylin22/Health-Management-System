# Requirement Analysis

## 1. Project Background

現代人生活步調加快，許多人因工作與課業繁忙而忽略自身健康狀況，導致體重管理、運動習慣及健康數據缺乏長期追蹤與紀錄。
目前市面上雖有許多健康相關應用程式，但功能較分散，使用者往往需要同時使用多個系統進行管理。因此，本專案希望建立一套健康管理系統，能夠整合個人健康資訊、運動紀錄與健康分析功能，讓使用者能更方便進行健康管理。

---

## 2. Problem Statement

目前面臨以下問題：

1. 缺乏集中式健康資料管理平台。
2. 無法有效追蹤體重與 BMI 變化。
3. 運動紀錄容易遺失或缺乏統計分析。
4. 缺乏長期健康數據分析功能。

---

## 3. Project Objectives

本系統之目標如下：

1. 建立會員註冊與登入機制。
2. 提供健康資料紀錄功能。
3. 提供運動紀錄管理功能。
4. 自動計算 BMI 指數。
5. 提供健康數據分析與查詢功能。
6. 建立完整個人健康資料庫。

---

## 4. Stakeholders

### Primary User

* 一般使用者
* 健康管理需求者
* 運動愛好者

### Secondary User

* 系統管理員

---

## 5. System Scope

### Included Functions

* 會員管理
* 健康資料管理
* BMI 計算
* 運動紀錄管理
* 健康分析

### Excluded Functions

* 醫療診斷功能
* 線上掛號功能
* 保險理賠功能

---

## 6. Expected Benefits

* 提升個人健康管理效率。
* 提供長期健康數據追蹤。
* 協助建立規律運動習慣。
* 提高使用者健康意識。
